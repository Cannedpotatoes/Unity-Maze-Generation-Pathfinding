# Unity 迷宫生成与寻路系统

## 功能特性

- 支持自定义尺寸的迷宫可视化生成（但在迷宫大于150*150时，由于Unity方面的优化问题，程序很有可能崩溃）
- 自动寻找并确定可用的起点和终点
- 迷宫生成和两种寻路算法的可视化展示
- 必要参数实时显示和调整、视角自由移动
- 迷宫生成部分采用深度优先搜索构建DFS生成树，迷宫寻路部分提供了两种算法，DFS深度优先搜索和BFS广度优先搜索

## 开始

1. 克隆项目：`git clone https://github.com/你的用户名/仓库名.git`
2. 用Unity 2021.3+打开项目，编辑器编者使用的均为Unity 6000.2.2f1
3. 打开场景：`Assets/Scenes/SampleScene.unity`
